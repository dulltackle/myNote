# 对症选点

- 手部：C1-7
- 头面部疾病：C1-7、T1-3
- 心脏：C1-7，T1-5，胸骨柄
- 颈椎修复注射：颈椎椎板和关节突关节
- 肺功能：T1-5和胸骨柄
- 内脏疾病：T6-12
- 肝胆病：T7-10
- 带状疱疹：T1-5
- 胃肠病：T6-12
- 胰腺+脾：T5-9
- 糖尿病：框架矫正+脊柱减压
- 牛皮癣
- 盆腔疾病：T12-L4+S2-4+骶骨翼
- 腰椎修复注射：腰椎椎板和关节突关节，腰3-5横突，侧隐窝，椎间外口
- 免疫性疾病：胸骨柄
- 消化和肝胆：胸骨体
- 脑脊液循环：枕骨
- 眼：枕骨
- 耳：枕骨+乳突尖松解
- 男性功能：八髎穴、阴部神经、耻骨结节、阴茎根部、骶骨翼背面
- 痛经：耻骨结节
- 结构三角：两侧骶髂关节+腰椎棘突
